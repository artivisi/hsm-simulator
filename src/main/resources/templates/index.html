About This Simulator

A comprehensive platform for simulating Hardware Security Module (HSM) operations in banking environments. This educational tool demonstrates cryptographic key management, including key ceremony processes, terminal key distribution (TMK, TPK, TSK), and inter-bank key exchange operations (ZMK, ZPK, ZSK) for the four-party card processing model.

Features include Shamir's Secret Sharing with 2-of-3 threshold scheme, complete key hierarchy management, key rotation with audit trail, and zone key exchange tracking for secure inter-bank communication.

Educational Use Only

This is a simulator designed for educational and training purposes only. DO NOT use in production environments. The cryptographic operations are simplified demonstrations and do not meet production security requirements. No actual sensitive data should be processed through this system.
