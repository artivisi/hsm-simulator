Contribution Error

Unable to Process Contribution

Invalid or expired contribution token

This could happen for several reasons:

  • • The contribution link has expired
  • • You have already submitted your contribution
  • • The ceremony is no longer accepting contributions
  • • The contribution link is invalid or corrupted

What Should You Do?

  1. Contact the ceremony administrator for assistance
  2. Request a new contribution link if needed
  3. Verify you're using the most recent link sent to you
  4. Check if you've already completed your contribution

If you believe this is an error, please contact support
