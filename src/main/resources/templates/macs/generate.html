What is a MAC?

  • Message Authentication Code: Cryptographic checksum that verifies message integrity and authenticity
  • TSK (Terminal Security Key): Used for MACs on terminal-to-switch messages (ATM/POS transactions)
  • ZSK (Zone Session Key): Used for MACs on inter-bank messages (network transactions)
  • ISO 9797-1 Alg 3: Retail MAC standard used in payment industry
  • HMAC-SHA256: Modern MAC using SHA-256 hash function
