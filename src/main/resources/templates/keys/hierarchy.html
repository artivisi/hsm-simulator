Key Types

Terminal Key Hierarchy

  • TMK - Terminal Master Key: Root key for terminal operations
  • TPK - Terminal PIN Key: Derived from TMK, encrypts PINs
  • TSK - Terminal Security Key: Derived from TMK, generates MACs

Zone Key Hierarchy

  • ZMK - Zone Master Key: Root key for inter-bank communication
  • ZPK - Zone PIN Key: Derived from ZMK, inter-bank PIN encryption
  • ZSK - Zone Session Key: Derived from ZMK, session data encryption
