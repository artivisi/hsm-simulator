Re-encrypt PIN from one key to another for secure transmission or storage

Why PIN Translation?

  • LMK → TPK: Translate from storage key to terminal key for ATM/POS transmission
  • LMK → ZPK: Translate from storage key to zone key for inter-bank transmission
  • TPK → ZPK: Translate from terminal key to zone key for transaction routing
  • Security: PIN never appears in clear text during translation
